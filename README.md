### Инструкция по установке и запуску

```bash
# 1. Клонирование репозитория
git clone https://github.com/regina-ms/only-digital.git
cd only-digital

# 2. Установка зависимостей
npm install

# 3. Сборка проекта для продакшена
npm run build:prod

# 4. Локальный запуск собранного проекта
npx serve build
